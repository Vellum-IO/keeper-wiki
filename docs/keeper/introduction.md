# Introduction

## Project Goal

The project goal is to provide a simple ui which allows customers to create postgres clusters with Replication, automatic updates, Disaster Recovery and automatic Backups on an enterprise ready scale.

## Capabilities

- Simple UI
- PG database cluster creation
- Metrics for External (Users of the service) and internal (us as developers)

## Functions and Features

### Non-Functional:

- Good UX
- Simple UI
- Simple Point of Entry
- Secure

### Functional

- Cluster Creation
- Credential rotation
- Message Queue for Failure recovery
- Creation of multiple clusters without lags
- Fast and Comparable IOPS to self hosted versions
